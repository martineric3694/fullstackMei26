🚀 MULTIFINANCE IT TALENT INCUBATOR

From "Ready to Learn" (Lulusan Bootcamp) to "Ready to Deliver" (Elite Talent)

🎯 Tujuan Utama

Mencetak 7-10 orang talenta elit yang benar-benar siap masuk ke project multifinance nyata dengan memadukan Pemahaman Bisnis dan Keahlian Teknis Spesifik.

Fase 1: Matrikulasi & Business Foundation

Membekali peserta dengan pemahaman SDLC, SQL, API, dan konteks bisnis end-to-end multifinance sebelum masuk ke koding spesifik. (Durasi: ~2.5 Minggu)

Modul 1-2 | 1.5 Hari

Industri & Loan Lifecycle

Overview multifinance, perbedaan produk, Revenue Model, dan alur end-to-end: Lead → Approval → Disbursement → Collection.

Modul 3-4 | 1.5 Hari

Terminologi & Core System

Memahami OTR, DP, Tenor, NPL, DPD. Pengenalan LOS, Credit Scoring, Disbursement System, hingga Recovery.

Modul 5-6 | 1.5 Hari

Financial Calculation & Impact Analysis

Logika bunga flat, angsuran, bucket collection. Studi kasus analisis dampak perubahan DP dari 20% ke 15% pada sistem.

Modul 7-9 | 1.5-2 Hari

Data, Risk & Simulation

SQL reporting (Approval, NPL), Fraud awareness, dan Mini-simulasi penambahan Grace Period pada sistem.

Fase 2: Stream Spesialisasi (3 Minggu)

💻 DEVELOPER STREAM

Target: Junior Dev, App Support, Enhancement

1. Enterprise Architecture
2 Hari | Layered Arch, Repository Pattern, SOLID
2. .NET Framework Fundamentals
3 Hari | ASP.NET Core, MVC, API, Entity Framework
3. Database for Enterprise
3 Hari | SP, View, Trigger, Transaksi (Approval Case), ERD
4. API Integration & Debugging
4 Hari | REST, JWT, Retry, Root Cause Analysis, Log Tracing
5. Secure Coding & Mini Project
3 Hari | SQL Injection, XSS, Build Loan Application API

🧪 QA / TESTER STREAM

Target: Manual Tester, UAT Tester, Jr. Automation

1. Software Testing Foundation
2 Hari | SDLC, STLC, Defect Lifecycle
2. Test Design & Case Writing
4 Hari | Boundary Value, Decision Table, Pos/Neg Scenario
3. UAT & API Testing
4 Hari | Requirement Review, Postman, Assertion
4. SQL for Tester & Bug Mgt
2 Hari | Data Validation, Root Cause, Defect Report
5. Automation & Mini Project
6 Hari | Playwright (Locator, Action), POM, Test Loan App

Fase 3: The Ultimate Differentiator

🔄 Multifinance Project Simulation (1 Minggu)

Penggabungan Developer dan QA untuk menjalankan miniatur siklus proyek sesungguhnya.

Hari 1 Requirement Given
Hari 2 Developer Coding
Hari 3 QA Create Test Case
Hari 4 UAT Execution
Hari 5 Bug Fixing (Dev)
Hari 6 Retest (QA)
Hari 7 Production Deploy
"Lulusan tidak hanya menguasai teknologi, tetapi mengalami miniatur siklus proyek multifinance yang sesungguhnya."
